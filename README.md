# team_manager

## Setting environment:
1. **Clone Repository from GitHub** <br />
2. **Crete and Activate Virtual Environment** <br />
3. **Install:** <br />
    `pip install Django` <br />
    `pip install djangorestframework` <br />
    
## App setup:
1. **Run server** <br />
    `python manage.py runserver` <br />
